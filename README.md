## UCAS网络登录

命令行的方式登录UCAS网络

```
usage: python[2/3] ucas.py [OPTION [params]]
    login [-p] [username] [password]
            login to ucas network, e.g. python ucas.py user123 pass123
            -p, --prefix: add '发\' before username
    logout
            logout from ucas network
    find
            find a user automatically

```
